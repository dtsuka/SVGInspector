# SVG Inspector

VS Code 上で SVG ファイルを視覚的に編集できる拡張機能です。
Webview を利用した専用エディタにより、レイヤー構造の確認や属性の編集を直感的に行うことができます。

## 機能

- **SVG プレビュー**: SVG をレンダリングして表示します。マウス操作によるパン（移動）やズーム（拡大縮小）が可能です。
- **レイヤーツリー**: SVG の要素（`<g>`, `<path>`, `<rect>` など）を階層構造で表示します。
    - **ドラッグ&ドロップ**: 要素をドラッグして並び替えが可能です。
    - **表示/非表示**: 目のアイコンをクリックして要素の表示/非表示を切り替えられます。
- **高度な選択機能**:
    - **キャンバス同期**: プレビュー上の要素をクリックすると、レイヤーツリーでも該当要素が選択されます。
    - **複数選択**: Shift/Cmd(Ctrl)キーを使って複数の要素を選択できます。
- **グループ化**: 複数選択した要素をグループ化（`<g>`タグで囲む）したり、グループ解除したりできます。
- **属性編集**: 選択した要素の属性（`id`, `class`, `fill`, `stroke` など）を一覧表示し、編集・追加・削除が可能です。
- **リアルタイム同期**: 編集内容はリアルタイムで SVG ファイルに反映されます（保存操作でファイルに書き込まれます）。

## 使い方

1. VS Code で `.svg` ファイルを開きます。
2. 自動的に SVG Inspector が起動します。
   - テキストエディタとして開きたい場合は、ファイルタブの右クリックメニューから「テキストエディタで開く」などを選択してください。
   - または、コマンドパレットから「Reopen Editor With...」を選択し、エディタを切り替えることができます。

## 開発者向け情報

この拡張機能は以下の構成で開発されています。

- **Extension Host**: TypeScript
- **Webview UI**: React + Vite

### ビルド手順

初回セットアップ:

```bash
pnpm install
cd webview-ui && pnpm install
```

Webview のビルド:

```bash
cd webview-ui && pnpm build
```

デバッグ実行:
`cd webview-ui && pnpm dev` を実行してから VS Code の「実行とデバッグ」サイドバーから "Run Extension" を選択して実行してください。

##  拡張機能のパッケージ化 (.vsix の作成)
